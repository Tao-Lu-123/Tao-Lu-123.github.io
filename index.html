About

Project

  • GPU Inference Acceleration for Large Language Models
    Designed a high-throughput GPU inference approach for LLMs based on moderately unstructured sparse weight matrices. Built an optimized sparse matrix-matrix multiplication (SpMM) kernel and corresponding scheduling strategy to improve memory efficiency and end-to-end inference throughput. [Code]
  • ZPrize (3rd Place)
    Implemented a high-performance GPU-based multi-scalar multiplication algorithm for ZKPs. Achieved a 1.67x speedup over the reference implementation. [Code]
